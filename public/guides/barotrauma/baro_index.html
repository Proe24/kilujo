☰Setup & load order (read first)

Three setup rules that break things if ignored

  • Install LuaCs via the launch-option method. Neurotrauma, Performance Fix and Enhanced Immersion's Lua features need it. See the LuaCs page.
  • DynamicEuropa requires a fresh campaign and must never be removed mid-campaign — it kills the map. See the DynamicEuropa page.
  • Client vs server side. Everyone subscribes to the same list, runs the LuaCs installer, and the host picks LuaForBarotrauma as the Server Executable.

How the list works

Barotrauma's list runs top → bottom. For this pack, follow each author's "place X above Y" notes — the rule that matters is that a mod higher in the list keeps priority over the lower mods it overlaps with. You only need to preserve the relative positions below.
