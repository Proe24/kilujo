Additive Beacon Content

ADV Beacon Stations

A large pack (~54 stations, several with shuttles) added alongside the vanilla beacons to make beacon-activation missions far less repetitive. Works in any game mode, can be added to an in-progress campaign, and is broadly compatible.

Mission / Event / Environment Campaign-safe to add anytime By PsykokiN
What it adds
~54 new beacon station layouts (some include a docked shuttle) into the beacon rotation.
Load position
Additive content band; keep below Barotraumatic. Confirmed compatible with DynamicEuropa and "pretty much anything else."
Activation
Same loop as vanilla: repair the hull/equipment, rewire as needed, restore power, pump the water, activate the sonar.

⚠Station-specific notes

  • Mines: the areas around stations ADVBS-037 and ADVBS-041 are mined — the mines explode if you get too close. Approach carefully.
  • Station 52 puzzle: beyond normal maintenance, you must manage the power supply via the power distributor so the junction boxes are adequately powered (the author confirms it's "a bit tricky").
  • Some stations are built with shuttles, which take longer to make — expect occasional larger/more involved layouts.

⚑How to use it

Nothing to install beyond enabling the mod — the new stations simply appear when beacon-activation missions generate. It mixes freely with the other beacon packs in this list (Beacons Extended, Even More, Facehurt's) since they all just add to the same pool.
