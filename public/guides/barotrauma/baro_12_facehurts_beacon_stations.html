Additive Beacon Content

Facehurt's Beacon Stations

Four cinematic beacon stations themed as underwater biodomes with glass exteriors. Balanced around vanilla using vanilla-only tags and assets — and the secure cabinets can hand you a small early-game windfall.

Mission / Beacon station Vanilla-balanced By Facehurt
What it adds
4 beacon stations with a glass biodome aesthetic (more planned).
Balance
Uses only vanilla tags/assets, so it slots into vanilla-balanced campaigns cleanly.

⚑What to expect

  • Each station has two Secure Steel Cabinets with auto-fill, which can spawn starter weapons + ammo — a handy early windfall (commenters found this slightly generous; the author considered a later-spawning balanced variant).
  • Loot is otherwise on the generous side; junction boxes spawn pre-damaged, so expect to rewire/repair as usual.
  • Stacks freely with the other beacon packs here.
