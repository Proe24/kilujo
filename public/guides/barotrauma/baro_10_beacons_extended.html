Additive Beacon Content

Beacons Extended

A curated collection of ~56 beacon stations from various authors, dropped into the beacon pool to reduce repetition. Purely additive — it makes no changes to the vanilla game at all.

Environment No vanilla changes Sibling to Shipwrecks Extended
What it adds
~56 beacons from many contributors, added to the activation rotation.
Compatibility
Compatible with all basic beacon mods unless filenames collide; it changes nothing vanilla.
Important
It does NOT add missions (and currently no valve-mechanic beacons) — just beacon variety. If you're chasing a "rearm weapons platform"-type mission, that's a different mod.

⚑How to use it

Enable it and the new beacons appear in normal beacon-activation missions — same activation loop as vanilla (repair, rewire, power, pump, sonar). It stacks cleanly with the other beacon packs here. Pairs with the same team's Shipwrecks Extended.
